CSS 选择器

CSS 选择器用于在 CSS 中指定样式匹配的元素集合，同时 CSS 选择器也可用于 JavaScript 选择页面元素来使用。

学习 CSS 前首先需要能够熟悉使用 CSS 选择器。

目前选择器共有以下几种：

基础选择器

比较常用的为基础选择器。

这类选择器最为基本，请务必掌握它的使用。

属性选择器

对于元素属性可以使用属性选择器。

这类选择器在设置表单元素 <input> 时，为选择不同类型的输入框时较为常用，可以选择性了解它的使用。

属性选择器的性能比基础选择器差，如果条件允许，应尽量减少其使用。

伪类选择器

对于元素的不同状态或顺序可以使用伪类选择器。

这类选择器通常用在控制用户不同的交互行为下（如鼠标经过、鼠标按）元素的展现样式，或控制一系列元素中的首个、末位或奇偶个不同的样式（如表格隔行不同背景颜色）。
