CSS 过渡

CSS 过渡是 CSS3 新增的可以使部分样式属性变化时平滑过渡的特性。

在制作交互界面时，过渡通常运用在 :hover 等伪类使样式属性发生变化时，让用户鼠标经过时元素样式出现平滑过渡的效果。

有一个著名的项目很好得展示了以上效果：hover.css。

CSS 过渡语法

transition主要包含四个属性值：

  • transition-property：允许过渡的属性名称
  • transition-duration：过渡持续时间
  • transition-timing-function：过渡时间函数
  • transition-delay：过渡开始延迟时间
transition: [<'transition-property'> || <'transition-duration'> || <'transition-timing-function'> || <'transition-delay'>
         [, [<'transition-property'> || <'transition-duration'> || <'transition-timing-function'> || <'transition-delay'>]]*

过渡属性名称 transition-property

在过渡中，允许同时定义多个可过渡属性，中间用逗号分隔。

如：

.demo {
  transition: transform .3s ease-in-out, opacity .2s linear .1s
}

注意：

  • 还有一个特殊的属性 all 代表所有可过渡的属性，慎用 all，因为部分属性的过渡变化会造成浏览器的不断回流（Reflow），极大影响性能。详见 CSS 动画 - 高性能 CSS 动画。

过渡时间函数 transition-timing-function

过渡时间函数用于描述过渡变化快慢。

有以下几个取值方式：

曲线变化

平滑地过渡

  • linear： 线性过渡。等同于贝塞尔曲线 cubic-bezier(0, 0, 1, 1)
  • ease： 平滑过渡。等同于贝塞尔曲线 cubic-bezier(0.25, 0.1, 0.25, 1)
  • ease-in： 由慢到快。等同于贝塞尔曲线 cubic-bezier(0.42, 0, 1, 1)
  • ease-out： 由快到慢。等同于贝塞尔曲线 cubic-bezier(0, 0, 0.58, 1)
  • ease-in-out： 由慢到快再到慢。等同于贝塞尔曲线 cubic-bezier(0.42, 0, 0.58, 1)
  • cubic-bezier(<number>, <number>, <number>, <number>)： 特定的贝塞尔曲线类型

工具：

  • cubic-bezier

技巧：

  • cubic-bezier 指定的值可以为负数，也可以大于1，意味着，你可以利用这个特性做出回弹效果的动画，如：cubic-bezier(.6, -0.9, .5, 1.6)

帧动画

这种很特殊的时间函数可以做出帧动画的效果，例如 这个效果。

  • step-start： 等同于 steps(1, start)
  • step-end： 等同于 steps(1, end)
  • steps(<integer>[, [ start | end ] ]?)： 接受两个参数的步进函数。第一个参数必须为正整数，指定函数的步数。第二个参数取值可以是 start 或 end，指定每一步的值发生变化的时间点。第二个参数是可选的，默认值为 end。
