CSS 定位

块级元素和行级元素

div、h1、p 这类元素默认占用页面中一整行，他们在页面中占据整块的空间，可以给他们设置宽度与高度，这类元素属于块级元素(block)。

span、a、strong 这类元素的使用并不占据一整行，他们属于行内元素，宽度和高度的设置对他们无效，这类元素属于行级元素(inline)。

通过对 display 的设置，可以改变一个元素的默认类型，如：

strong {
    display: block;
}

但是在一种情况下，即使没有进行显式定义，也会创建块级元素。这种情况发生在把行级元素添加到块级元素的兄弟位置。即使没有把这些文本定义为段落，它也会被当作段落对待。

<div>
    some text <!-- 此处自动产生一个块级元素 -->
    <p>Some more text.</p>
</div>

CSS 定位机制

CSS 有三种基本的定位机制：文档流、浮动和绝对定位。

我们将一个页面从上到下视为一个流，称为文档流。除非专门指定，否则所有元素都在文档流中定位。也就是说，文档流中的元素的位置由元素在 HTML 中的位置决定。

块级元素从上到下排列。

行级元素从左到右排列，排列不下时产生换行。

position 属性

通过使用 position 属性，我们可以选择 4 种不同类型的定位。

static 静态定位

CSS 中的默认定位方式。

行级元素生成一个矩形框，作为文档流的一部分。行内元素则会创建一个或多个行框，置于其父元素中。

relative 相对定位

元素由当前定位的位置偏移某个距离。

元素仍保持其未定位前的形状，它原本所占的空间仍保留。

注意：

  • 相对定位是元素依然在文档流中，因为元素的位置是相对于它在文档流中的位置。

absolute 绝对定位

元素从文档流完全抽离，并相对于其父元素中非 static 定位方式的容器进行定位。

元素原先在正常文档流中所占的空间会取消。

使用绝对定位的元素不论原来它是否为行级元素，此时均改为块级元素。

fixed 固定定位

元素的表现类似于绝对定位，不过其是相对窗口进行绝对定位，即固定在窗口的某个位置。

同绝对定位相同，元素原先在正常文档流中所占的空间会取消。

使用绝对定位的元素不论原来它是否为行级元素，此时均改为块级元素。

练习（石英钟 - CSS 定位）

从这一章节开始，我们开始尝试自己编写一个石英钟，最终效果如下：

我们马上就可以利用到已了解到的知识。

首先，在 CodePen 中创建一个 New Pen，粘贴如下的代码。

HTML:

<div id="container">
  <div id="clock">
  </div>
</div>

CSS:

#clock {
  position: absolute;
  top: 50%;
  left: 50%;
}

#container {
  position: absolute;
  top: 0;
  bottom: 0;
  left: 0;
  right: 0;
  background: #232425;
  font: 14px/1.4 "Helvetica Neue", Helvetica, Arial, sans-serif;
}

#clock {
  width: 26em;
  height: 26em;
  margin: -13em 0 0 -13em;
  background: #f9f9f9;
}

说明

练习中的几个知识点：

几个较为常用的尺寸单位：

  • px 像素，计算机屏幕上的一个点
  • % 相对父容器的尺寸，仅在父容器尺寸确定的情况下有效
  • em 相对当前元素中字号（font-size）的比例。由于字号可以由父元素继承，因此可以达到通过修改一处 font-size，改变页面所有元素大小的效果。

负 margin

是的，你没有看错。CSS 盒模型中的外边距 margin 是可以为负的。

这是一个非常有用的特性，可以使 static 定位的元素依然可以进行偏移；absolute 与 fixed 时，常用于处理居中。

在我们的练习中，#clock 元素是 absolute 定位，与容器上方、左边边距均为 50%，再通过设置负的一半的元素尺寸，补充元素的偏移位置，让元素达到上下左右居中的目的。
