CSS 背景

在之前的代码中，我们已经见到了使用 background 或 background-color 设置背景色。

其实 CSS 中的背景可以实现非常强大的功能。

CSS 背景包含如下这些属性：

  • background-color ：规定要使用的背景颜色。
  • background-image ：规定要使用的背景图像。
  • background-repeat ：规定如何重复背景图像。
  • background-attachment ：规定背景图像是否固定或者随着页面的其余部分滚动。
  • background-position ：规定背景图像的位置。

CSS3 中新增了如下属性：

  • background-size ：规定背景图片的尺寸。
  • background-origin ：规定背景图片的定位区域。
  • background-clip ：规定背景的绘制区域。

以上这些属性，可以分开来单独指定，也可以使用一个 background 来全部囊括。

p {
  background: #00FF00 url(bg.jpg) no-repeat fixed top;
}

以上属性用空格分隔分别代表了：

  • background-color：背景色（背景图未覆盖到的位置）
  • background-image：背景图 URL
  • background-repeat：no-repeat 不重复；repeat-x 横向重复；repeat-y 纵向重复；、repeat 横向纵向均重复（默认值）
  • background-attachment：scroll 背景图像会随着页面其余部分的滚动而移动（默认值）； fixed 当页面的其余部分滚动时，背景图像不会移动。
  • background-position：背景定位，详见此处

以上属性都很好理解，在这里重点介绍 CSS3 中新增的背景相关属性。

background-size

背景尺寸，这个属性是 CSS3 中增加的，可以允许通过如下几种方式指定背景的尺寸：

  • 百分比（50%）
  • 具体尺寸（100px）
  • 填充到元素区域（cover）
  • 包含到元素区域（contain）

其中下面两种非常实用：

cover 允许背景图与元素区域长宽比例不同，超出元素范围时填充满元素，然后多余部分自动裁剪。

contain 允许背景图与元素区域长宽比例不同，超出元素范围时整体被包含到元素区域，等比例缩放完整显示，元素区域剩余位置用指定的背景色填充。

background-origin

规定背景图片的定位区域，与盒模型中指定盒尺寸计算方式类似。

有以下几种定位方式：

  • padding-box ：背景图像相对于内边距框来定位。
  • border-box ：背景图像相对于边框盒来定位。
  • content-box ：背景图像相对于内容框来定位。

background-clip

规定背景的绘制区域，与 background-origin 不同之处在于 background-origin 指定的是相对定位方式，background-clip 指定的是背景绘制在什么框内。

  • border-box ：背景被裁剪到边框盒。
  • padding-box ：背景被裁剪到内边距框。
  • content-box ：背景被裁剪到内容框。
