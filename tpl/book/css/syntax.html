CSS 语法

CSS 语法非常简单，由两个主要的部分构成：

  • 选择器
  • 一条或多条样式声明
selector {
    property1: value1; /* 注释内容 */
    property2: value2;
    property3: value3;
}

即：

选择器 {
    样式属性1: 样式值1;
    样式属性2: 样式值2;
    样式属性3: 样式值3;
}

CSS 的工作原理

根据选择器匹配 HTML 元素，将为匹配到的元素设置样式。

子元素将从父元素继承部分可以被继承的样式。

加入 CSS 的方法

  • 使用 <link> 标签引入。

      <link rel="stylesheet" type="text/css" href="style.css" />

  • 使用 <style> 标签内嵌。

      <style>
      body {
          background: #eee;
          color: #333;
      }
  </style>

  • 元素上使用 style 属性指定。

      <div style="background: #eee; color: #333;">...</div>

为了项目的工程化管理，我们统一使用 <link> 标签引入 CSS 样式文件。
