CSS 阴影

CSS 中有两种阴影，分别是文本阴影 text-shadow 和盒阴影 box-shadow。

文本阴影顾名思义就是为文本内容添加阴影，而盒阴影是 CSS3 加入的新特性，可以为任意盒子添加阴影。曾经需要贴图的阴影如今也可以用 CSS 来绘制了。

CSS 阴影语法

这两种阴影的语法其实非常非常类似，均支持为目标添加一个或用逗号隔开的多个阴影。

文本阴影

text-shadow: <length> <length> <length>?||<color>

几个参数按顺序分别是X轴偏移量、Y轴偏移量、阴影模糊半径、阴影颜色。

阴影模糊半径可选，但值只能是为正，如果值为 0 时，表示阴影不具有模糊效果，值越大阴影就越模糊。

.demo {
  text-shadow: 10px 8px 6px #777;
}

DEMO

为文本设置多个阴影时会有很多有趣的效果。

.demo {
  text-shadow: 2px 2px 0 #fff, 4px 4px 0 #777;
}

DEMO

.demo {
  color: #fff;
  text-shadow: 1px 1px 0 #000, 1px -1px 0 #000, -1px 1px 0 #000, -1px -1px 0 #000;
}

盒阴影

box-shadow: <inset>? <length> <length> <length>? <length>?||<color>

几个参数按顺序分别是投影方式、X轴偏移量、Y轴偏移量、阴影模糊半径、阴影扩展半径、阴影颜色。

我们可以注意到，它比文本阴影增加了投影方式和阴影扩展半径。

投影方式有两种：外阴影和内阴影，默认为外阴影如果需要使用内阴影，则在该阴影声明前加上 inset。

阴影扩展半径可选，作用是可以整体扩展或阴影的范围，因此其取值可以为负。

.demo {
  box-shadow: 2px 8px 6px #777;
}

内阴影的效果

.demo {
  box-shadow: inset 2px 8px 6px #777;
}

同样，设置多个阴影时会有很多有趣的效果。

.demo {
  box-shadow: 3px 0 3px #f00, -3px 0 3px #0f0, 0 -3px 3px #f80, 0 3px 3px #0ff;
}


.demo {
  box-shadow: 0 0 0 3px #f00, 0 0 0 6px #0f0, 0 0 0 9px #f80, 0 0 0 12px #0ff;
}

练习（石英钟 - CSS 阴影）

我们开始为石英钟添加阴影的部分。

石英钟外壳

我们为石英钟分别添加整体的外阴影。

外壳上的高光则使用亮色的内阴影来绘制。

#clock
  box-shadow: .3em .5em 2em #333, inset 0 0 2em rgba(#ddd, .4),
              inset 0 -.2em .2em rgba(#000, .3)

石英钟表盘外框

在 #clock::before 伪元素为石英钟的表盘外框增加外阴影，使之有突出的感觉。

#clock
  &::before
    box-shadow: .1em .3em 1em rgba(#333, .2)

石英钟表盘

在 #clock::after 伪元素为石英钟的表盘增加内阴影，作为刚刚在外框上添加外阴影的另外一边阴影。

同时再添加一个比较透明的亮色内阴影作为表盘玻璃的高光。

#clock
  &::after
    box-shadow: inset rgba(#000, .2) .2em .2em 1em,
                inset 12em 14em 0 -10em rgba(#fff, .3)

指针

时针、分针、秒针也需要添加阴影，使之显得立体突出。

.hand
  box-shadow: rgba(#000, .8) 0 0 .2em

秒针中心的圆需要单独添加外阴影。

#second
  .outer
    &::after
      box-shadow: rgba(#000, .8) 0 0 .2em
