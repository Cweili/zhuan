HTML 标签

  • 标签由英文尖括号 < 和 > 括起来，如 <html> 就是一个标签。

  • html中的标签一般都是成对出现的，分开始标签和结束标签。结束标签比开始标签多了一个 /。

    如：

      <p>段落</p>
  <div>层</div>
  <span>行内文本</span>

  • 也有部分标签是没有另一对的，我们称之为自闭合标签，可以在标签自身结尾添加 / 结束标签。

    如：

      <img src="图片.jpg" alt="图片标题" />
  <input type="text" name="textInput" placeholder="文本输入框" />

  • 标签与标签之间是可以嵌套的，但先后顺序必须保持一致，如：<div> 里嵌套 <p>，那么 </p> 必须放在 </div> 的前面。

    如：

      <div>
      <p>
          <span>行内文本</span>
      </p>
  </div>

  • HTML 标签不区分大小写，<h1> 和 <H1> 是一样的，但我们要求为全部小写，这是为了便于模板管理以及其他框架的解析。

  • 标签中可以指定属性 <a href="超链接地址">超链接</a>。

  • 标签可以添加 id 属性来指定该标签在页面内唯一的 id。
