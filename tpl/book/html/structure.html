HTML 基本结构

通常一个 HTML 文档结构为：

<html>
    <head>
        <!-- 文档头部 -->
    </head>
    <body>
        <!-- 文档内容 -->
    </body>
</html>

  • <html></html> 称为根标签，所有的网页标签都在 <html></html> 中。

  • <head> 标签用于定义文档的头部。头部元素常见有“<title> 页面标题”、“<script> 脚本”、“<style> 样式”、“<link> 链接”、 “<meta> 元信息”等标签。

  • 在 <body> 和</body> 标签之间的内容是网页的主要内容，如 <h1>、<p>、<a>、<img> 等内容标签，在这里的内容会在浏览器中显示出来。

  • HTML 的注释使用 <!-- 注释内容 -->。
