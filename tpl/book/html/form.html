表单

表单是 HTML 中重要的组件，需要获取由用户填写的内容时，我们就需要用到表单。

一个表单通常有以下几种标签：

  • <form> 定义供用户输入的表单
  • <input> 定义输入域
  • <textarea> 定义文本域 (一个多行的输入控件)
  • <label> 定义一个控制的标签
  • <fieldset> 定义域
  • <legend> 定义域的标题
  • <select> 定义一个选择列表
  • <optgroup> 定义选项组
  • <option> 定义下拉列表中的选项
  • <button> 定义一个按钮

注意：

  • <form>、<input>、<textarea>、<select> 标签需要包含 name 属性，来指定该表单元素的字段名称。

  • <input> 标签需要给出 type 属性，来指定类型，常用的类型如下：

    • text：文本输入框
    • password：密码输入框
    • hidden：隐藏输入框
    • number：数值输入框
    • date：日期输入框
    • time：时间输入框
    • datetime-local：日期时间输入框
    • email：E-mail 输入框
    • file：文件输入框
    • radio：单选框（通过设置相同的 name 属性关联）
    • checkbox：多选框（通过设置相同的 name 属性关联）

  • <button> 标签也需要给出 type 属性，来指定类型，常用的类型如下：

    • button：普通按钮
    • submit：提交表单按钮
    • reset：重置（清空）表单按钮

  • <label> 标签如果没有包含子节点 <input> 或 <textarea>，则需要包含 for 属性来指向对应表单元素，for 属性的值为对应元素的 id

    如：

      <label for="exampleInputText">文本</label>
  <input type="text" name="exampleInputText" id="exampleInputText">

  • 通过指定 value 属性，可以指定该表单元素默认值。

    如：

      <input type="text" name="exampleInputText" id="exampleInputText" value="example">

  • 部分类型输入框可以通过指定 placeholder 属性来指定用户未输入时，输入框内默认显示内容。

  • 可以为表单元素指定 readonly 属性，使表单元素只读；或 disabled 属性，使表单元素被禁用：

    如：

      <input type="text" name="exampleInputText" id="exampleInputText" value="example" readonly>
  <input type="text" name="exampleInputText" id="exampleInputText" disabled>

  • 可以为表单元素指定 required 属性，使表单元素要求必须填写：

    如：

      <input type="text" name="exampleInputText" id="exampleInputText" required>

  • 不同的表单元素有不同的特殊的属性，如文本类型输入框可以指定 length 限制用户的输入长度；数值和日期类型输入框可以指定 min、max、step 限制用户的输入的最小值、最大值、增减时的间隔等，详见 input 标签。
